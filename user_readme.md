# Как пользоваться этим проектом

Это не готовое приложение — это заготовка для разработки **с ИИ Claude Code**. Внутри лежат
агенты (готовые ИИ-помощники) и правила для них. Вы словами описываете задачу в чате — ИИ
планирует, пишет код, проверяет его. Настоящий код вашего проекта появится в этой же папке
по мере работы.

## Что нужно перед стартом

1. Установленный **Claude Code**. Если ещё не установлен:
   ```
   npm install -g @anthropic-ai/claude-code
   ```
   (нужен Node.js; если его нет — сначала поставьте Node.js с официального сайта).
2. Больше ничего. Python и остальные зависимости подтянутся по ходу работы, если понадобятся.

## Первый запуск — 3 минуты

1. Откройте терминал **в этой папке**.
2. Запустите:
   ```
   claude
   ```
3. Напишите словами, что хотите сделать. Пример:
   ```
   @planner хочу телеграм-бота, который раз в день присылает погоду
   ```
4. ИИ покажет план. Прочитайте и напишите `Реализуй по плану` — или сначала попросите
   что-то поправить в плане.
5. Когда код готов:
   ```
   @code-reviewer проверь
   ```

Это весь цикл. Для следующей фичи или правки просто повторяете шаги 3–5 в том же чате
или в новом.

## Не знаете, с чего начать?

Напишите:
```
@advisor [опишите, что хотите сделать, или что сейчас происходит]
```
Advisor подскажет, кого позвать и в каком порядке — коротко, без лишнего.

## Что-то сломалось?

Просто опишите проблему словами, как коллеге:
```
кнопка "Отправить" не работает
```
ИИ сам найдёт причину в коде и почини́т. После фикса попросите `@code-reviewer проверь`.

## Сессия закрылась — как продолжить

Если план был сложный, `@planner` сохраняет его в файл `docs/plans/<дата>-<название>.md`.
В новом чате напишите:
```
Продолжи по docs/plans/<файл>
```
ИИ прочитает файл, увидит готовые шаги и продолжит с того места, где остановились.

## Что не стоит удалять без нужды

| Папка / файл | Зачем |
|---|---|
| `agents/` | Инструкции для ИИ-агентов — `@planner`, `@code-reviewer` и т.д. |
| `.claude/` | Настройки и навыки Claude Code |
| `CLAUDE.md` | Общие правила проекта — ИИ читает их автоматически, ничего вставлять в чат не нужно |
| `docs/harness/` | Более подробные гайды (см. ниже) |
| `docs/plans/` | Сохранённые планы фич — история того, что и как делалось |

Свой код можно класть куда угодно ещё (например `src/`) — агенты сами предложат структуру,
когда дойдёт до дела.
