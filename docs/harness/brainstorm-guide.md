# Мозговой штурм простыми словами

Это инструмент, где **несколько ИИ обсуждают вашу задачу между собой**, а потом один из
них **подводит итог в том виде, который вам нужен** (список идей, план или готовая
HTML-страница). Вам не нужно ничего программировать — вы ставите задачу словами и
получаете идеи и готовый результат.

## Зачем это нужно

Один ИИ может что-то упустить. Здесь их несколько, и они **спорят и дополняют друг
друга**: один предлагает, второй критикует и предлагает лучше, третий собирает всё
лучшее вместе. Как совещание команды — только за пару минут.

## Как запустить

Откройте **Git Bash** в папке проекта и введите:

```bash
python scripts/brainstorm-live.py
```

Можно сразу написать задачу в кавычках:

```bash
python scripts/brainstorm-live.py "придумай, как удобно скачивать картинки с сайта по категориям"
```

Откроется страница в браузере, где всё будет происходить у вас на глазах.

## Что у вас спросят (по шагам)

1. **Что обдумать?** — опишите задачу своими словами (одно обязательное поле).
2. **Необязательные уточнения** — несколько вопросов (контекст, что уже известно, критерии
   успеха, на чём сделать акцент, что не рассматривать). Отвечать не обязательно — жмите
   **Enter**, чтобы пропустить любой из них. Чем больше расскажете, тем точнее будут идеи.
3. **Как назвать папку?** — придумайте короткое имя. Результаты сложатся в папку
   `docs/brainstorm/<ваше-имя>_<дата>/` (дата добавится сама).
4. **Кто участвует** — для трёх ролей выбираете ИИ и «модель»:
   - **Начинающий** — предлагает идеи;
   - **Отвечающий** — критикует и улучшает;
   - **Подводящий итоги** — собирает финальный результат.

   Доступны **Claude**, **Codex** и **GLM**. Если не знаете, что выбрать — берите первый
   вариант в каждом списке, это разумные значения по умолчанию.
5. **Глубина размышления** — насколько тщательно думать. Проще так: выше уровень = умнее
   и обстоятельнее, но дольше и дороже. Для начала подойдёт `medium`.
6. **Сколько раундов** — сколько раз они обменяются репликами (1 раунд = предложение +
   критика). Начните с 1–2.

После этого просто наблюдаете за обсуждением в браузере.

## Что вы получите

Всё лежит в папке `docs/brainstorm/<ваше-имя>_<дата>/`:

| Файл | Что это |
|------|---------|
| **result.html** | Готовый итог — **всегда красивая веб-страница**: открывается двойным кликом, с аккуратным оформлением и простыми графиками там, где это уместно |
| **conversation.html** | Вся беседа ИИ, красиво оформленная — можно перечитать позже |
| **transcript.md** | То же самое текстом |

## Полезно знать

- **Итог — всегда удобная для чтения страница** (`result.html`): без технического мусора,
  с заголовками, списками и простыми графиками/шкалами, если они уместны для темы.
- **Всё, что вы видите** — и во время работы, и в результатах — сделано **читаемым для
  человека**. Сырого кода и служебных строк там нет.
- **Можно смешивать ИИ как угодно**: например, начинает Codex, отвечает GLM, а итог
  подводит Claude. Или всё делает один и тот же ИИ.
- Каждый запуск — отдельная папка, старые результаты не перезаписываются.

Технические детали (модели, уровни размышления, требования) — в [scripts.md](./scripts.md).
