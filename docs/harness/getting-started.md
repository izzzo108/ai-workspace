# Getting Started

Код пишет ИИ. Вы пишете задачи в чате.

## Первые 10 минут

**В Claude Code (терминал):**
1. Открыть терминал в папке проекта
2. Запустить `claude`
3. Написать в чате: `@planner хочу телеграм-бота, уведомления раз в день`
4. Обсудить план или написать: `Реализуй по плану`
5. После — `@code-reviewer проверь`

Вместо шага 3 можно использовать готовую slash-команду, например `/feature-development`.

---

## Как использовать эту болванку

| Ситуация | Что делать |
|----------|------------|
| **Новый проект** | Скопировать всю папку `ai-workspace` → дать новое имя → открыть терминал в этой папке → запустить `claude` |
| **Старый проект** | Скопировать `agents/`, `.claude/`, `docs/harness/`, `CLAUDE.md` внутрь папки старого проекта |

После копирования сразу можно работать: `@planner` → реализация → `@code-reviewer`.

Код проекта появится рядом — например в `src/` или там, куда укажет planner.

---

## Не знаешь с чего начать?

```
@advisor [опиши что хочешь сделать или что происходит]
```

Advisor — агент-советник. Скажи ему ситуацию — он ответит кого звать и в каком порядке. Коротко и просто.

---

## Ваш цикл (4 шага)

| # | Вы пишете | Что происходит |
|---|-----------|----------------|
| 1 | `@planner хочу телеграм-бота, уведомления раз в день` | Planner составляет план |
| 2 | *(по желанию)* обсудить план или `Реализуй по плану` | Уточняете или ИИ пишет код |
| 3 | `@code-reviewer проверь` | Проверка кода |
| 4 | `Исправь критичное` | Правки |

Всё в одном чате — ИИ видит план выше и пишет код под него.

На шаге 2 (реализация) TDD включается **автоматически** — упоминать его не нужно, это часть пайплайна, а не отдельный шаг, который вы просите.

Сложный проект (магазин + оплата + …) → сначала `@architect`, потом `@planner`. См. [agents-new-project.md](./agents-new-project.md).

---

## Сессия прервалась — как продолжить

`@planner` сохраняет план не только в чат, но и в файл: `docs/plans/<дата>-<название>.md`. У каждой фазы там статус ⬜/🔄/✅.

Если чат закрылся или сжался (compaction), просто откройте новый чат и напишите:
```
Продолжи по docs/plans/<файл>
```
Claude прочитает план, увидит готовые фазы (✅) и продолжит со следующей. Подробнее — [docs/plans/README.md](../plans/README.md).

## Папки (не трогать без нужды)

| Папка | Зачем |
|-------|-------|
| `agents/` | Агенты — `@planner`, `@code-reviewer` и др. |
| `.claude/skills/` | Подсказки для Claude Code — включаются сами по типу файла |
| `docs/harness/` | Эти гайды |
| `docs/plans/` | Сохранённые планы фич от planner — со статусом фаз |

Skills редактировать не нужно, они включаются автоматически.
